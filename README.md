# ADA Assignments Repository

This repository contains Python scripts, Jupyter notebooks, and submitted PDF documents for ADA assignment.

## Folder Contents

### Python Scripts
- `assignment_2.py`: Search, sorting, knapsack, Fibonacci (with timing comparison setup), and TSP algorithms.
- `assignment_3.py`: Graph representations and graph algorithms (BFS, DFS, topological sort, Dijkstra, Prim).
- `assignment_4.py`: Backtracking and branch-and-bound scheduling plus naive string matching and timing test.

### Jupyter Notebooks
- `Task_01.ipynb`: Notebook for Task 1 work.
- `Task_02.ipynb`: Notebook for Task 2 work.
- `Task_03.ipynb`: Notebook for Task 3 work.
- `Task_04.ipynb`: Notebook for Task 4 work.

### PDF Files
- `assignment_2_satyam.pdf`: Assignment 2 submission/report.
- `assignment_3_satyam.pdf`: Assignment 3 submission/report.
- `assignment_4_satyam.pdf`: Assignment 4 submission/report.
- `SATYAM_2401420022.pdf`: Additional submitted document.
- `satyam_capstone_project_ada.pdf`: Capstone/project PDF document.

## How To Run

1. Install Python 3.9+.
2. Install required packages:

```bash
pip install numpy matplotlib
```

3. Run scripts:

```bash
python assignment_2.py
python assignment_3.py
python assignment_4.py
```

## Notes

- Script outputs are printed directly to the terminal.
- Some tasks may include example/demo inputs hardcoded in the files.
